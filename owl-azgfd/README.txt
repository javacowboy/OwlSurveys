After unzipping the code zip file:
1. Copy database Excel spreadsheet files to the "data" folder.
2. Double click the owl-azgfd jar file.
3. Check the generated spreadsheet file named: azgfd_output.xlsx

Troubleshooting:
Instead of double clicking the jar file:
1. Open a command line terminal.
2. cd (change directory) to the folder where the owl-azgfd jar file is located.
3. type: java -jar owl-azgfd.jar
4. As the code executes, any errors should log/print to the command line terminal.
